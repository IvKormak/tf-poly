# Polynomial net

Библиотека для генерации и использования полиномиальных нейронных сетей на Python

## Установка

Зависимости:

matplotlib==3.4.3

numpy==1.21.3

scipy==1.8.0

pandas==1.4.2

sympy==1.11.1


## Демо

Запустите demo.ipynb и запустите все ячейки последовательно.

## Использование

Импортируйте файл polynomial.py в свой проект
